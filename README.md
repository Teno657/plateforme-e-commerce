# Plateforme e-commerce

## Description
Cette plateforme e-commerce permet la vente de produits en ligne.
Elle offre aux clients la possibilité de consulter les produits,
passer des commandes et effectuer des paiements, tandis que
l’administrateur gère les produits, les commandes et les utilisateurs.

## Fonctionnalités
- Inscription et authentification des utilisateurs
- Catalogue de produits
- Panier d’achat
- Gestion des commandes
- Gestion des utilisateurs
- Tableau de bord administrateur

## Technologies utilisées
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
- XAMPP

## Installation
1. Installer XAMPP
2. Démarrer Apache et MySQL
3. Importer la base de données
4. Copier le projet dans le dossier `htdocs`
5. Accéder à l’application via le navigateur
